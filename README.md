# Kötbér Kalkulátor 📱

A simple web-based tool to calculate **early termination penalties** for 2-year fixed telecom contracts.

---

## What it does

Enter a few details and instantly find out how much penalty a customer would owe if they terminate their contract early:

- 📅 Contract start date + planned termination date
- 📦 Package type — **Normál S** or **Korlátlan M**
- 🌐 Internet package selection
- 💰 Full (non-discounted) prices for voice and data

The calculator shows a full breakdown and the **total penalty amount**.

---

## How to use

Just open the link — no install, no login needed:

👉 **[Open Calculator](https://yourusername.github.io/kotber-kalkulator)**

Works on desktop and mobile.

---

## How the penalty is calculated

> **(Full price − Discounted price) × Number of months used**

Applied separately for voice and data, then summed.
